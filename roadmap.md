# 🗂️ FocusTabs - Roadmap
A minimalist, local-first goal management app designed with FAANG-level architecture and a 2025 aesthetic.

---

## ✅ Core Architecture
- [x] React Native + Expo SDK 52
- [x] Expo Router + _layout.tsx system
- [x] Strict TypeScript (%100 type safety)
- [x] Zustand-based store infrastructure
- [x] SQLite for persistent offline storage
- [x] Local-first strategy (no initial auth)
- [x] ESLint + Prettier setup
- [x] Git + Branch strategy + Modular PR policy
- [ ] CI/CD pipeline with GitHub Actions or EAS

---

## 🧠 System Design
- [x] Clean Architecture structure
- [x] Modular folder setup (`src/components`, `store`, `screens`, `hooks`, `lib`)
- [x] `dailyGoalsStore.ts` + daily reset logic
